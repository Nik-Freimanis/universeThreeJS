
# Universe Landing

![UniverseApp](https://i.ibb.co/R9mCXCL/universe.png)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Nik-Freimanis/universeThreeJS
```

Go to the project directory

```bash
  cd universeThreeJS
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- ThreeJS
- framer-motion
- react-tilt
- adaptive layouts with tailwind
